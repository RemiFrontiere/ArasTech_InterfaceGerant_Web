### Create project
You need angular-cli
```sh
  npm install -g angular-cli
  ng new PROJECT_NAME
```

### Install dependencies - [In your
This command install all dependencies listed in package.json
```sh
  npm install
```

### Start dev server
```sh
  ng serve / npm start

  ( Si port déjà occupé )
  ng serve --port[CHOIXDUPORT]
```

### Build
```sh
  ng build / npm run build
```

### Firebase
```sh
firebase deploy
```
